{{- $p := . -}} {{- $pages := $p.Pages -}} {{- if $pages -}} {{- $paginator := .Paginate $pages -}} {{- if $paginator -}} {{- $prev := $paginator.Prev -}} {{- $next := $paginator.Next -}} {{- if or $prev $next -}}
{{- if $prev -}}
Previous {{ $prev.Title }}
{{- end -}} {{- if $next -}}
Next {{ $next.Title }}
{{- end -}}
{{- end -}} {{- end -}} {{- else -}} {{- $prev := .PrevInSection -}} {{- $next := .NextInSection -}} {{- if or $prev $next -}}
    {{- if $prev -}}
    Previous: {{ $prev.Title }}
    {{- end -}} {{- if $next -}}
    Next: {{ $next.Title }}
    {{- end -}}
{{- end -}} {{- end -}}
